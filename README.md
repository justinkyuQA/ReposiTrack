# ReposiTrack

**Every repository. One dashboard.**

ReposiTrack is a workspace dashboard that discovers, inventories, and tracks software repositories.

It monitors project health, Git status, documentation, releases, screenshots, and development progress from one place.

---

## Mission

Track the health, progress, releases, and readiness of every repository from one dashboard.

---

## Features

- Repository discovery
- Git status detection
- Branch detection
- Documentation audit
- Screenshot detection
- Health scoring
- Workspace dashboard
- Markdown reports
- JSON reports

---

## Commands

python -m repositrack version

python -m repositrack repo .

python -m repositrack dashboard ~/projects

python -m repositrack report ~/projects

---

## Philosophy

ReposiTrack doesn't replace your repositories.

It watches them.
